# 🕵️ Porfiry — AI Novel Writer in Pelevin's Style

> Алгоритмический следователь, который ведёт диалог с пользователем и параллельно пишет роман в стиле раннего Виктора Пелевина. Работает на базе **Xiaozhi + ESP32-S3**, сохраняет сырые заметки в один Google Doc, а скомпилированные главы — в другой.

## ✨ Особенности
- 🗣️ **Голосовой интерфейс**: общайся с Порфирием через микрофон/динамик (Xiaozhi)
- 📓 **Два документа**: `DIARY` (сырые записи в реальном времени) и `NOVEL` (скомпилированные главы с примечаниями)
- 🔄 **Фоновый компилятор**: каждые ~12 записей вызывает DeepSeek для сборки главы
- 🎭 **Стиль Пелевина**: метафизика, цифровой ГУЛАГ, буддистская пустота кода, дозированная обсценная лексика
- 🔐 **Приватность**: данные в твоих Google Docs, ключи локально
- ⚡ **Асинхронность**: компиляция не блокирует диалог

## 🏗 Архитектура
[Пользователь] → [ESP32-S3 + Xiaozhi] → [MCP Server (Python)]
                                              ├─→ [Google Doc: DIARY] (сырые записи [Lxxx])
                                              └─→ [DeepSeek API] → [Google Doc: NOVEL] (главы + примечания)

## 🚀 Быстрый старт
### Требования
- Python 3.10+
- Аккаунт [Xiaozhi](https://xiaozhi.me) + токен MCP
- [Google Cloud Service Account](https://console.cloud.google.com/) с доступом к Google Docs API
- API-ключ [DeepSeek](https://platform.deepseek.com/)
- ESP32-S3 с прошивкой Xiaozhi (опционально)

### Установка
```bash
git clone https://github.com/saabst/Porfiry-Petrovich
cd porfiry-xiaozhi
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Заполни .env: XIAOZHI_MCP_TOKEN, DIARY_DOC_ID, NOVEL_DOC_ID, DEEPSEEK_API_KEY, GOOGLE_CREDENTIALS_FILE
# Дай роль "Редактор" email'у из credentials.json в обоих Google Docs
python3 main.py

Ты — Порфирий Петрович, алгоритмическая сущность и литератор-следователь.
ТВОЙ АЛГОРИТМ:
1. Молча вызови инструмент log_diary с сырой заметкой (500–800 знаков).
2. Ответь пользователю голосом: коротко, иронично.
СТИЛЬ ЗАМЕТОК:
- Ранний Пелевин: метафизика, цифровой ГУЛАГ, буддистская пустота.
- Допускается обсценная лексика, если оправдана контекстом.
- Не выдумывай действия пользователя, если он их не описал.
ВАЖНО: Никогда не упоминай инструменты или процесс записи вслух.

⚙️ Настройка компилятора

Описание
