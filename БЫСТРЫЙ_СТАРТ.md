# ⚡ Быстрый старт с Порфирием Петровичем

## 🚀 5-минутная установка

### 1. Клонирование и настройка
```bash
# Клонируйте репозиторий
git clone https://github.com/saabst/Porfiry-Petrovich.git
cd Porfiry-Petrovich

# Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Установите зависимости
pip install -r requirements.txt
```

### 2. Базовая конфигурация
```bash
# Создайте файл .env
cat > .env << EOF
XIAOZHI_MCP_TOKEN=ваш_токен
DEEPSEEK_API_KEY=ваш_ключ
COMPILE_THRESHOLD=5  # Для быстрого тестирования
USE_IMAGES=true
USE_PDF=true
LOG_LEVEL=INFO
EOF
```

### 3. Тестовый запуск
```bash
# Запустите сервер
python3 main.py

# ОПЦИОНАЛЬНО ДЛЯ ТЕСТА В другом терминале отправьте тестовую заметку
python3 -c "
import asyncio
import websockets
import json

async def test():
    uri = 'wss://api.xiaozhi.me/mcp/?token=ВАШ_ТОКЕН'
    async with websockets.connect(uri) as ws:
        await ws.send(json.dumps({
            'jsonrpc': '2.0',
            'method': 'tools/call',
            'params': {
                'name': 'log_diary',
                'arguments': {
                    'text': 'Тестовая заметка для быстрого старта',
                    'themes': ['тест', 'старт']
                }
            },
            'id': 1
        }))
        print('Заметка отправлена')

asyncio.run(test())
"
```

## 📊 Быстрая проверка состояния

### Скрипт проверки здоровья
```bash
#!/bin/bash
# health_check.sh

echo "🔍 Проверка здоровья Порфирия..."

# 1. Проверка Python
echo -n "Python: "
python3 --version

# 2. Проверка виртуального окружения
echo -n "Виртуальное окружение: "
if [ -d "venv" ]; then
    echo "✅ Найдено"
else
    echo "❌ Не найдено"
fi

# 3. Проверка .env
echo -n "Файл .env: "
if [ -f ".env" ]; then
    echo "✅ Найден"
    source .env
    echo "  Токен: ${XIAOZHI_MCP_TOKEN:0:10}..."
    echo "  API ключ: ${DEEPSEEK_API_KEY:0:10}..."
else
    echo "❌ Не найден"
fi

# 4. Проверка состояния
echo -n "Файл состояния: "
if [ -f "story_state.json" ]; then
    echo "✅ Найден"
    chapters=$(grep -o '"chapter_count": [0-9]*' story_state.json | grep -o '[0-9]*')
    entries=$(grep -o '"entries_count": [0-9]*' story_state.json | grep -o '[0-9]*')
    echo "  Глав: $chapters"
    echo "  Заметок в кэше: $entries"
else
    echo "❌ Не найден"
fi

# 5. Проверка output
echo -n "Папка output: "
if [ -d "output" ]; then
    echo "✅ Найдена"
    ls -la output/
else
    echo "❌ Не найдена"
fi

echo "✅ Проверка завершена"
```

1. **Проверьте токены**: верные ли API ключи?
2. **Проверьте интернет**: есть ли подключение?
